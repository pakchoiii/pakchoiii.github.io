NAND Flash：我们身边的“数据仓库”

● 什么是NAND Flash？
它是一种非易失性存储介质，就像一个不需要插电也能保存文件的“保险柜”，即使断电，数据也不会丢失。

● 无处不在的身影
U盘、SD卡、固态硬盘(SSD)、手机存储，从消费电子到工业设备，它几乎无处不在。

● 为什么选择它？
速度快、体积小、功耗低、抗震性强，彻底改变了我们存储和使用数据的方式。

数据的最小容器：浮栅晶体管 (Floating-gate MOSFET)

● 结构：单一控制栅极，如同普通的门
● 功能：仅控制电流通断，类似物理开关
● 特点：易失性，断电/撤去电压即恢复初始状态

● 结构：控制栅极 + 悬浮”浮栅”结构
● 功能：通断控制 + 捕获电子并长期保存
● 特点：非易失性，电子一旦被”陷阱”捕获很难逃逸

核心差异：浮栅（Floating-gate）的存在让晶体管从“临时开关”进化为“永久记忆体”

NAND的“仓库布局”：从货架到整个仓库群

存储单元的层级映射：Cell → Page → Block → Plane → LUN

● Cell (存储单元)：最小的存储单位（浮栅晶体管）。好比一个货架。 ● Page (页)：由多个Cell组成，是最小的读写单位。好比一个抽屉。 ● Block (块)：由多个Page组成，是最小的擦除单位。好比一个独立的储物间。 ● Plane (平面)：由多个Block组成。好比一个楼层。 ● LUN (逻辑单元号)：由一个或多个Plane组成，是芯片级别的逻辑单元。好比一栋楼。
